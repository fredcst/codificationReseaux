# README3 - Programme "codeWordsP2"

Le programme codeWordsP2 affiche en écran les codeWords pour le polynôme:

p2(x)=(x+1)(x^4+x+1)

avec: n=8

La liste est avec séparateurs de colonnes et newLines pour LATEX.

## Utilisation du programme

### Compiler avec:

```bash
make all
```

### Démarrer avec:

```bash
./codeWordsP2
```
